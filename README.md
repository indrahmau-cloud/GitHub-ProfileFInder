# 🔍 GitHub Profile Finder

A sleek, responsive web application that allows users to search for GitHub profiles and view key statistics like followers, following, public repos, and recent repositories in real-time.


## ✨ Features
- **Real-time Search:** Fetch user data directly from the GitHub REST API.
- **Dynamic UI:** Displays profile avatars, bio, location, and social links.
- **Repo Highlights:** Automatically lists the top 5 most recent repositories with links.
- **Error Handling:** Graceful alerts for "User Not Found" or connectivity issues.
- **Responsive Design:** Optimized for both desktop and mobile viewing.

## 🛠️ Built With
- **HTML5:** Semantic structure for accessibility.
- **CSS3:** Custom styling and layout (Flexbox/Grid).
- **JavaScript (ES6+):** Logic for API fetching and DOM manipulation.
- **GitHub API:** The data source for profile information.

## 📖 How to Use
1. Enter a valid GitHub username into the search bar.
2. Press "Enter" or click the search icon.
3. View the user's statistics and click on repository names to visit them directly.
